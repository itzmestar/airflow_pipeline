# airflow_pipeline
ETL data pipeline example using airflow

## Run steps

1. clone repo
```commandline
git clone https://github.com/itzmestar/airflow_pipeline
```
2. Initialize airflow
```commandline
cd airflow_pipeline
docker-compose up airflow-init
```

3. Start Airflow container
```commandline
docker-compose up
```

4. check container status
```commandline
docker ps
```
